# 配置说明

## 采用技术
- 前端框架：vue3
- 页面路由：vue-router
- 状态管理库：pinia
- Ajax实现：axios
- 数据mock：json-server

## 安装与运行
```bash
 git clone 仓库地址
 cd 仓库名
 npm install
 npm run dev
 ```
